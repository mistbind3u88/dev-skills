# AGENTS.md

## このリポジトリの目的

特定のプロジェクトや業務領域に依存しない、汎用的な開発ワークフロースキルを管理する。

## スキルの追加・変更に関する制約

### 含めてよいもの

- git / GitHub の操作に関するスキル
- Claude Code のセットアップや設定に関するスキル
- コードレビュー、リント、フォーマットなど汎用的な開発プロセスのスキル
- 言語やフレームワークに依存しない汎用ツール連携のスキル

### 含めてはいけないもの

- **業務領域固有の知識やロジック** — 特定プロジェクトのドメインモデル、ビジネスルール、社内用語への依存
- **秘匿情報** — API キー、トークン、認証情報、社内 URL、内部システム名
- **特定組織のインフラ情報** — CI/CD パイプラインの詳細設定、デプロイ先のホスト名、内部ネットワーク構成
- **特定リポジトリの構造に依存するスキル** — 特定のディレクトリ構成やファイル命名規則を前提とするもの

### 判断基準

スキルを追加・変更する際は以下を確認すること:

1. **公開リポジトリである** — push される内容はすべて公開される前提で書く
2. **汎用性** — git/GitHub を使う任意のプロジェクトで利用できるか
3. **独立性** — 特定のリポジトリをクローンしなくても単体で動作するか
