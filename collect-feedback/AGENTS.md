# collect-feedback

## 前提ツール

- [gh](https://cli.github.com/) — `brew install gh`

## 分類の判断基準

### 対応済みの判断

以下の場合は「対応済み」と判断する。

- 指摘内容が現在のコードに反映されている
- コメント者自身が解決済みと返答している
- 議論の結果「対応不要」「現状でOK」と合意されている

### 要対応の判断

以下の場合は「要対応」と判断する。

- 修正依頼に対して返答はあるがコードが未修正
- 「〜した方がよい」に対して合意はあるが未実装
- 合意された方針変更が反映されていない
